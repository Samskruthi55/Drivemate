# Hire a Driver – UI/UX Mobile App Design

This repository contains the complete **UI/UX design** for a mobile application called **Hire a Driver**, designed to make hiring professional drivers simple, fast, and reliable.

## Project Overview
**Hire a Driver** is a service-based mobile app where users can:
- Select vehicle type (2-Wheeler / 4-Wheeler)
- Choose hire duration
- Find nearby available drivers
- View detailed driver profiles
- Book drivers and complete payments smoothly

The design focuses on **simplicity, clarity, and a modern gradient-based UI** to enhance user experience.

##  Design Goals
- Clean and intuitive user flow
- Minimal and professional UI
- Easy navigation for all user types
- Visually appealing gradient backgrounds
- Mobile-first design approach

## Tools & Platform
- **Design Tool:** Figma  
- **Device Type:** Android (Mobile – Compact)
- **Design Style:** Minimal, Modern, Gradient UI

## Screens Designed
- Login Screen  
- Home Screen  
- Driver Requirements Screen  
- Driver List Screen  
- Driver Profile Screen  
- Booking Summary Screen  
- Payment Screen  
- Payment Success Screen
